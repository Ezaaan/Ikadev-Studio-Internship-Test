## Delivery Game
### How to Run
Run ```Delivery Guy.exe ``` in the ```build``` directory
### How to Play
**Movement**  
```A``` - Move left  
```D``` - Move right  
```W``` - Jump    
**Action**  
```K``` - Pick up item / Give item
